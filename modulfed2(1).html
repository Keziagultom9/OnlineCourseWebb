Pemrogramnan Web

Course / Front-End Developer / Pemula

Setelah mengetahui tentang anatomi dan bagaimana website bekerja, pada kelas ini kita akan banyak mempelajari dasar HTML, CSS, dan JavaScript yang menjadi inti dari pengembangan website.

Pada sub-modul Pendahuluan HTML dan Pendalaman HTML kita akan mempelajari rangkaian penggunaan HTML pada website. Mulai dari pembuatan berkas HTML, mengenal dan menggunakan elemen, serta menampilkan gambar dan penyusunan layout. Kemudian pada sub-modul Pendahuluan Styling dan Pendalaman Styling kita akan mempelajari penerapan styling pada website untuk mempercantik dan membantu dalam penyusunan website. Sampai sub-modul ini, kita akan membuat sebuah halaman website yang menarik.

Pada sub-modul Responsive Layout menggunakan Flexbox, kita akan belajar bagaimana cara menyusun layout yang responsif menggunakan flexbox. Selain itu juga terdapat praktik membuat sebuah layout dasar kalkulator pada website.

Sedangkan JavaScript akan dibahas pada sub-modul JavaScript dan DOM (Document Object Model). Di sana kita akan belajar dasar-dasar dari bahasa pemrograman JavaScript, pengenalan syntax, penggunaan variabel, tipe data yang ada, iterator, dll. Selain itu kita juga akan belajar bagaimana cara memanipulasi “DOM” dengan menggunakan JavaScript sehingga dapat membuat website yang interaktif.

Pada sub-modul Web Storage, kita akan mempelajari cara mengelola sebuah data atau objek pada local storage ataupun session storage dengan memanfaatkan Web Storage. Sampai sub-modul ini, kita akan membuat sebuah proyek kalkulator sederhana dengan memanfaatkan Web Storage untuk menyimpan riwayat perhitungannya.

Dan pada akhir pembelajaran, untuk lulus dari kelas ini, terdapat submission alias proyek akhir yang harus Anda kerjakan dan kirimkan. Kemudian proyek tersebut akan kami review guna menentukan kelulusan Anda. Sudah siap belajar dasar pemrograman website? Sebelum memulainya mari kita persiapkan beberapa tools yang akan digunakan pada kelas ini.
